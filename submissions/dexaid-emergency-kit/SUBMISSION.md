Project name: DexAid: Closed-Loop Five-Finger Emergency Kit Assembly in MuJoCo

Short description:
A five-finger dexterous hand autonomously assembles an emergency medication kit in MuJoCo: grasping a vial, twisting the cap over 240 degrees, placing the correct dose, inserting a syringe connector, closing the kit, and recovering from slip/disturbance using tactile feedback.

Judge-facing highlights:
- 20/20 deterministic trial success
- avg pose error <5mm
- cap rotation >240°
- 5–6N disturbance hold
- 5-finger high-DOF hand, free-body vial, frictional contacts, tactile/proprioceptive sensing
- one-command reproduction: `pip install -r requirements.txt && python demo.py`
